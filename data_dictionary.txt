This document contains a list of the data elements that are available to visitors using the 
"Download the entire data set" link on the AuditorSearch website.  The file available at 
this link contains data that was filed by registered accounting firms in Form AP.  The file is 
updated daily. 
 
The downloadable file is written in CSV (comma separated values) format.  Fields that 
contain commas or other reserved characters are delimited by quotation marks.  Certain 
fields which contain the content of repeating elements from Form AP are provided in 
field/value pairs using the JSON (JavaScript Object Notation) format. 
 
Dates and times are provided based on the U.S. Eastern time zone. 
 
The data reference tab contains a list of the data elements that are included in the 
downloadable data set.  The data elements are listed in the order that they appear in 
Form AP.  The column labelled "CSV Column #" indicates the order in which the data 
elements appear in the CSV file.  The column labelled "Excel Column" indicates the order 
in which the data elements typically appear when the file is opened in Microsoft Excel. 
 
Additional information about Form AP can be found on the Form AP resource page 
(https://pcaobus.org/Pages/form-ap-reporting-certain-audit-participants.aspx) and in the 
Staff Guidance— Form AP, Auditor Reporting of Certain Audit Participants and Related 
Voluntary Audit Report Disclosure Under AS 3101, Reports on Audited Financial 
Statement.
